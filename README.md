# Full-Stack Blog Application

This is a full-stack blog application built with **Laravel** for the backend and **React (Vite)** for the frontend.  
It provides authentication, role-based access control, and CRUD functionality for posts and categories.

---

## Tech Stack

### Backend
- Laravel 10
- MySQL
- Laravel Sanctum (API authentication)

### Frontend
- React
- Vite
- Axios
- React Router
- Tailwind CSS
